## 9. 收录 / 分类说明

- **分级标准**：每个一级分类下，按「使用场景」拆为 2～4 个二级分类（大的套小的），便于直接定位；
- **标注规则**：
  - ◎ 需梯子 / 代理才能访问
  - ★ 编辑精选 / 较稳定
- **跳过内容**（不在本导航收录）：
  - R18 漫画 / 同人志 / 视觉小说 / 番剧 / 视频站点（请通过对应官方或合规渠道获取）
  - 明显侵权 / 闭源商业破解资源链接
- **维护建议**：如发现新站点 / 站点失效，欢迎通过 Issue / PR 补充。
- **免责声明**：本导航仅做**资源索引**，不托管任何媒体文件。访问 / 下载 / 使用第三方资源时，请遵守当地法律法规与各项目协议。
